| Eigenschaften, Erkennungsmerkmale, Besonderheiten,&nbsp;(Gattungen): |
| --- |
| Geruch: | Neutral bis angenehm pilzig. |
| Geschmack: | Bitter, kratzend, herb, später pfeffrig scharf. |
| Hut: | 1-4 (6) cm Ø, ockergelb, später rotbräunlich nachdunkelnd, muschelig, klebrig mit gummiartiger Schicht, Rand wellig, gerieft. |
| Fleisch: | Hellbraun, dünn. |
| Stiel: | Seitlich stehend und&nbsp;zur Anwachsstelle zugespitzt,&nbsp;rotbräunlich, ockergelblich wie Hut gefärbt, voll, elastisch. |
| Lamellen: | Dicht stehend, gegabelt, klebrig, oft gesägt uneben, am Grunde mit Querlamellen. |
| Sporenpulverfarbe: | Weiß (2,5-3 x 1-2 µm, ellipsoid, Cheilozystiden 25-75 x 2,5-5 µm). |
| Vorkommen: | Auf Laubhölzer, gerne Eiche, Buche, Birke..., Folgezersetzer, ganzjährig, frisch häufiger im Spätherbst und Winter. |
| Gattung: | Zwergknäuelinge, Muschelseitlinge. |
| Verwechslungsgefahr: | Milder Zwergknäueling&nbsp;auf Nadelholz mit Rosatönen,&nbsp;Harziger Sägeblättling,&nbsp;Violettblättriger Zwergknäueling,&nbsp;Gallertfleischiges Stummelfüßchen. |
| Kommentar: | Pilz in ein Glas legen und bei Dunkelheit ansehen, er leuchtet in der Nacht (fluoreszierender Effekt)! |
