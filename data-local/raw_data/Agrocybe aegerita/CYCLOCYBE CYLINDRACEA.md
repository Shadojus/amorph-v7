| Eigenschaften, Erkennungsmerkmale, Besonderheiten, (Gattungen): |
| --- |
| Geruch: | Neutral, angenehm oder nach ranzigem Mehl. |
| Geschmack: | Mild. |
| Hut: | 3-8 (11) cm Ø, creme-weiß, ocker-bräunlich, grubig-runzelig, Haut bei Trockenheit aufreißend, Mitte meist dunkler braun, Rand oft heller. |
| Fleisch: | Weißlich, Stielbasis dunkler, bräunlich. |
| Stiel: | Weißlich, mit hängendem, oft mit flüchtigem weißlich bis orangebraunen Ring, dieser ist oft durch Sporenpulver bräunlich verfärbt. |
| Lamellen: | Weißlich, hell creme-weiß, alt zimtbraun, tabakbraun, sehr gedrängt stehend, Schneiden wellig, breit angewachsen bis schwach herablaufend, oft punktiert fleckend. |
| Sporenpulverfarbe: | Tabakbraun bis ockerbraun (8-11 x 4,4-5,7 µm, elliptisch). |
| Vorkommen: | An Laubholz, vorwiegend an Stümpfen oder Stämmen von Pappeln, Folgezersetzer, +/- auch parasitär, den Baum angreifend, wärmeliebend, gerne Weinanbaugebiete, büschelig, Frühling bis Spätherbst. |
| Gattung: | Ackerlinge. |
| Verwechslungsgefahr: | Voreilender Ackerling, Berindeter Seitling, Cremeweißer Seitling. |
| Kommentar: | Kann kultiviert werden. |
| Relativer Speisewert: | Schweiz: Marktfähig; DGfM: Positivliste Speisepilz; 123pilze: Relative Wertigkeit 1. |
| Wiki-Link: | https://de.wikipedia.org/wiki/Südlicher_Ackerling |
| Priorität: | 2 |
